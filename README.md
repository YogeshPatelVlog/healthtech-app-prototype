# HealthTech Mobile App – Prototype

## Objective
To design a scalable health-tech mobile application prototype for frontline healthcare workers.

## Tech Stack (Planned)
• Flutter
• Provider (State Management)
• Hive / SQLite (Local Storage)
• Camera APIs
• Modular Architecture

## Features
✓ Authentication (Dummy)
✓ Dashboard
✓ Patient Management
✓ Image Capture
✓ AI Result Simulation
✓ Records & History
✓ Settings (Theme)

## AI Result Simulation
AI output is mock-based (Normal / Needs Review / High Risk) with explainability.
Architecture allows future ML model or API integration.

## Scalability
• Feature-based modular design
• Offline-first approach
• Backend & AI pluggable
